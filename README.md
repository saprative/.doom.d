# Doom Emacs Windows Setup

Emacs Setup using Doom. Setup with Windows PC.

Installation 
============
1. Setup Doom Emacs 
2. Setup org file in the root directory of windows 
3. Setup sqlite3 in the C:/sqlite3

What you can do ? 
1. Todolist
2. Note taking 
3. Code Browsing 
   1. Python 
   2. NodeJS
   3. PHP 
   4. ReactJS
   5. Golang 
   6. Docker
